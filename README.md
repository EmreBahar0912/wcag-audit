### Testbevindingen
Uit de Lighthous test kwam naar voren dat de kleurcontrast op bepaalde plekken niet goed genoeg waren, interactieve elementen waren niet te onderscheiden van 'normale' elementen en de heading volgorde was niet op juiste volgorde. Voor de toegankelijkheid voor alle gebruikers is het zeer belangrijk om deze benoemde punten te verbeteren, zodat iedereen gebruik kan maken van de website.
